## Features

###### 配置
1. 可以在config.py里调整搜索结果的排数
2. 可以在config.py里调整预览图片的大小，越小越快，但越模糊。

###### 图片预览
1. 可以快速预览远程的图片，带宽占用低
2. 可以点击小图看到清晰原图
3. 支持预览文件夹路径，.list文件, .tsv文件和单个图片文件
4. 支持页数跳转
5. 支持路径跳转
6. 支持responsive layout
7. 支持图片统计分析


## How to use
1. git clone \<this repo\>
2. cd OmniViewer
3. pip install -r requirements.txt
4. python manage.py runserver
